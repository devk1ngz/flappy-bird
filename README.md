## 🚀 Giới thiệu  
Đây là một phiên bản clone của tựa game huyền thoại **Flappy Bird**, được xây dựng bằng Python và Pygame.  

## 🎮 Cách chơi  
- `Click chuột` để giúp chú chim bay lên.  
- Tránh va chạm với cột ống nước.  
- Cố gắng đạt điểm cao nhất!  

## 🛠 Công nghệ sử dụng  
- Python 3  
- Pygame
